inorder to run the program
1.Install apache server(wamp is an easy way to get,to check if you are doing it right)
http://www.wampserver.com/en/ 
2.Install python 2.7
https://www.python.org/downloads/release/python-2710/
3.Download and Install bs4 module
https://pypi.python.org/pypi/beautifulsoup4/4.3.2
4.Enable cgi support that you can do using the following youtube link(the method given in the link is for apache 2.2,if >2.4 the commands change)
https://www.youtube.com/watch?v=hv77IgkojfQ
5.Place the startup folder in www folder in wamp
6.open 127.0.0.1 select startup project at the bottom
7.Enter the product name and check out the various prices from amazon,snapdeal and flipkart
